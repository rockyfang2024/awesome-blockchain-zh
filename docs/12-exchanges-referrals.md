## 💼 内推资源（加密公司招聘）

> 币圈公司招人**主要靠内推**，公开投递竞争激烈。
> 以下是常用的内推信息源 + 重点公司招聘入口。

### 📱 内推信息聚合

| 平台 | 特点 |
|------|------|
| **LinkedIn** | 直接联系目标公司的在职员工，请他们内推 |
| **脉脉** | 中文区，主要互联网公司（部分区块链公司在上面招人） |
| **Twitter / X** | 关注 `#[company_name] hiring` 等标签 |
| **Refermentor / 内推鸭 / 职得** | 中文互联网内推平台 |
| **电鸭** | 中文远程 + 加密公司常见 |

### 微信 / Telegram 群

> ⚠️ 微信群/Telegram 群鱼龙混杂，**谨防付费内推骗局**：
> - 真内推不会收你钱（最多请你喝奶茶）
> - 真 HR 不会要求先打款"激活账户"
> - 警惕冒充"猎头"要求你提供身份证/银行卡/验证码

| 群类型 | 加入方式 | 质量 |
|--------|---------|------|
| 各交易所官方招聘群 | 关注目标公司公众号，加 HR 微信 | ⭐⭐⭐⭐ |
| 链节点 / 巴比特内推板块 | 论坛直接发帖 | ⭐⭐⭐ |
| 大 V 招聘 bot | Twitter 关注 | ⭐⭐ |
| 微信"币圈内推"类群 | 朋友邀请 | ⭐⭐（小心骗局） |

### 🎯 内推准备清单

1. **LinkedIn 头像 + 简介**：英文区必备
2. **GitHub 完整**：开源贡献 > 学历
3. **个人项目**：哪怕是 hackathon 项目
4. **Twitter / 推文**：英文圈习惯
5. **推荐信**：上一份工作的同事/老板
6. **技术写作**：博客 / 笔记 / Mirror / Mirror.xyz

### ⚠️ 求职避坑

| 🚩 红旗 | 说明 |
|--------|------|
| 付费内推 | **100% 骗局**，正规内推不收钱 |
| 招聘"操盘手" / "交易员" | 大概率是资金盘 |
| 承诺"稳定收益" + 让你入金 | 庞氏骗局 |
| 远程工作 + 高薪 + 不面试 | 可能是洗钱岗位 |
| "先打款激活账户" / "需要押金" | 100% 骗局 |
| 招聘"量化策略师"让你自己带资金 | 亏损你承担，盈利他要分成 |
