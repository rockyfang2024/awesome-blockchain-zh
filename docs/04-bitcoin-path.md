# ₿ Bitcoin 学习路径

> 这一章带你深入理解 Bitcoin —— 区块链最老、最稳、共识最强的公链。

## 🎯 必学顺序

1. **白皮书**（30 min）—— 中本聪的 9 页纸，**必读**
2. **Mastering Bitcoin**（1-2 周）—— 系统性教材
3. **源码**（1 周+）—— 看 bitcoin/bitcoin 真实实现
