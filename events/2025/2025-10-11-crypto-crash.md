# 2025-10-11 币圈史上最大规模爆仓事件

> **事件类型**: market_event（市场行情事件）

## 🔗 传导链（江卓尔分析）

```
10/11 05:20  BTC/ETH/山寨币 跌到最低
         │
         ▼
    传导 1：USDe 开始脱锚（05:20 略微 → 05:44 最低 $0.65）
    原因：循环贷套利者市价卖出 USDe 换 USDT
         │
         ▼
    传导 2：wbETH / bnSOL 开始脱锚（05:43~05:44）
    原因：USDe 暴跌 → USDe 循环贷爆仓 → 
         wbETH/bnSOL 抵押借贷账户被强平
```

### 关键反证：不是做市商爆仓

> 江卓尔排除"大做市/套利商爆仓"假说的关键证据：

- **做市/套利商都是 期货现货对冲**（做多现货 + 做空期货赚资金费率）
- 如果这类玩家爆仓，**必然出现 山寨币异常（现货/期货反向异动）**
- 5:43am 检查 DOGE 等山寨币：**usdt 现货和永续期货交易均正常**
- ⇒ **没有大做市商爆仓**

**真正爆仓的对象**：用 ETH/SOL 质押 → wbETH/bnSOL → 抵押借 USDT → 换 USDe 循环贷的**散户/矿工**。

## 🎰 循环贷杠杆机制（核心知识点）

### USDe 循环贷套利模型

```
第 1 步：USD → USDe（拿 12% 年化收益，活动期）
第 2 步：USDe 抵押 → 借 USDT（78% 抵押率）
第 3 步：USDT → USDe（再赚利息）
循环 4.45 次 = 4.45 倍杠杆
```

### wbETH 套利模型（更隐蔽）

```
10 万 ETH 借 6 万 USDT（60% 抵押率）

原成本：
  6 万 USDT 年利息 5.5% = 3,300 USDT

优化路径：
  10 万 ETH → 币安质押 → 拿 wbETH（2.49% POS 利息）
  wbETH 抵押 → 借 USDT
  收益 = 10万 × 2.49% = 2,490 USDT/年

实际利息 = 3,300 - 2,490 = 810 USDT/年
       = 年化 1.35%（从 5.5% 降到 1.35%）
```

**看起来很美好，但代价：**
- 10 万 wbETH 借 **1.13 万 USDT** 就爆仓（**币安强平率 91%**）
- 杠杆 ≈ **9 倍**
- wbETH 流动性差，一旦脱锚**无缓冲空间**

## 🏛️ 币安的责任

江卓尔认为币安有责任且已部分担责：

### 责任点

1. **wbETH / bnSOL 是币安质押发行的凭证**
   - ETH POS 收益 2.89%，币安分 0.5%，用户拿 2.39%
   - 币安有义务维持其流动性和锚定

2. **价格指数算法有问题**（已改进）
   - **旧算法**：80% wbETH/USDT + 20% wbETH/ETH
   - **新算法**：100% wbETH/ETH
   - 这个改动**只是小优化**，非根本解决

3. **根本问题：流动性不足**
   - wbETH/ETH 日交易深度仅 2000+ ETH
   - 无法防止下次再发生严重脱锚

### 币安已采取的措施

- ✅ **补偿** USDe / wbETH / bnSOL 异常交易的损失
- ✅ 补偿标准 = **低价卖出价 vs 11日 08:00 交易价的差价**
- ⚠️ 但"不足以完全防止下次脱锚"

### 江卓尔的建议

> **建议设置计算价值时 ETH/SOL 的汇率硬底（例如 0.8）**

意思是：计算抵押率时，wbETH 价值最多算 0.8 ETH，而不是市场价（0.2x ETH）。
这样用户被强平的阈值会更安全（10万wbETH 从借1.13万提高到借1.83万u爆仓）。

## 🎓 学习要点

### 对普通用户

1. **不要被表面收益迷惑**
   - 5.5% → 1.35% 的"优化"看起来很美
   - 代价是 **9 倍杠杆 + 流动性差 = 闪崩时 80% 价值蒸发**
2. **DeFi 循环贷 = 隐藏高杠杆**
   - 4.45x 循环 + 嵌套质押 = 实际杠杆可能 **20x+**
3. **wbETH 这类 LST 流动性陷阱**
   - 平时交易正常，但暴跌时**没人接盘**
   - 价格指数会被算法扭曲

### 对交易者

1. **关注宏观传导链**：中美关税 → 美股 → BTC → DeFi
2. **凌晨 4-6 点是亚洲-美国时差真空**：流动性最差，波动放大
3. **DeFi 嵌套产品 = 雷曼式连锁风险**
4. **稳定币脱锚（USDe 0.65）= 系统性风险信号**

### 对协议设计者 / 币安

1. **LST 价格指数应该加硬底**（江卓尔建议的 0.8 ETH）
2. **流动性深度必须预警** —— wbETH/ETH 日仅 2000 ETH 远不够
3. **抵押率参数要保守**：91% 强平率 + 8x 杠杆 = 太激进
4. **DeFi 嵌套的风险传染**：单一资产暴跌 → 多个协议连环爆

### 对监管 / 观察者

1. **DeFi 循环贷实质 = 监管盲区的高杠杆**
2. **CEX 发行的 LST（wbETH）有中心化风险**：算法和流动性都靠 CEX
3. **跨境事件传导**：地缘政治 → 美股 → 加密 → DeFi 全链条
